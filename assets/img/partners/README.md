# Partner logos

Drop each vendor's **official** logo here (only logos you're licensed to display),
using these exact filenames. `.svg` is preferred; `.png` also works if you update
the `src` extension in `index.html`.

| File                       | Brand                |
|----------------------------|----------------------|
| `cisco.svg`                | Cisco                |
| `microsoft.svg`            | Microsoft            |
| `crowdstrike.svg`          | CrowdStrike          |
| `paloaltonetworks.svg`     | Palo Alto Networks   |
| `sentinelone.svg`          | SentinelOne          |
| `beyondtrust.svg`          | BeyondTrust          |

Each logo is auto-normalized to a uniform height (28px) in the trust bar, shown in
grayscale and colorized on hover. Any brand without a file here falls back to a
clean text wordmark, so the bar never looks broken.
